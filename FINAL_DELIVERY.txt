   "Create a simple API service"
   ↓

   ↓
3. Select Preferences (Cards appear):
   • Purpose: API Service 🌐
   • Features: Error Handling ⚠️, Logging 📝
   • Performance: High-Performance 🚀
   • Testing: Comprehensive ✅
   • Documentation: Standard 📖
   ↓
4. Click "Generate Skill"
   ↓
5. Skill generated with:
   • Your description
   • Your preferences (automatically included)
   • Production-ready code
   • All files and documentation
   ↓
6. Download or view results
