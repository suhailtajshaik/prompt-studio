## Breaking Changes

⚠️ **For existing code that depends on old structure:**

Old JSON schema:
```json
{
  "name": "...",
  "skillmd": "...",
  "scripts": {...},
  "references": {...}
}
```

New JSON schema:
```json
{
  "name": "...",
  "language": "python",
  "main_code": "...",
  "main_file": "main.py",
  "dependency_file": "requirements.txt",
  "dependency_content": "...",
  "readme": "...",
  "example_file": "example.py",
  "example_code": "...",
  "test_file": "test.py",
  "test_code": "..."
}
```

**Migration:** Any downstream code expecting `skillmd` field should be updated to use `main_code` instead.
