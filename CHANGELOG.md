# Changelog — Prompt Studio

## [1.2.0] - 2026-03-14
### Added
- Agent Mode tab: 2-step LLM pipeline (Clarity Refine → Agent Decomposition)
- useAgentDecompose hook for multi-step agent planning
- AgentModeView component with agent cards and execution blueprint
- Copy Blueprint button for JSON export

## [1.1.0] - 2026-03-14
### Added
- Clarity Refine technique: intent extraction, noise removal, structured output
### Removed
- LocalAI provider (no longer supported)

## [1.0.0] - 2026-02-01
### Added
- Initial release: COSTAR, 6-Step, Markdown frameworks
- Techniques: Zero-Shot, Few-Shot, CoT, Role, Structured Output, Meta Prompting
- Provider support: Anthropic, Gemini, OpenRouter
- Dark/Light theme
